Usage Guidelines

Brand Conformance Rules

  • Primary Squiggle: Use the toothbrush squiggle from the logo as the main brand motif
  • Pattern Hierarchy: Light (3% opacity), Mid (6% opacity), Dark (12% opacity)
  • Pattern Usage: Light for subtle backgrounds, Mid for main content areas, Dark for footer/dark sections
  • Opacity Hierarchy: Main elements 0.6-0.8, decorative elements 0.2-0.4, background patterns 0.03-0.12
  • Stroke Width: 0.6px for light patterns, 1px for mid patterns, 1.2px for dark patterns
  • Color Consistency: Always use WD Gold (#E6A444) for squiggle elements
  • Animation Timing: Use gentle easing (cubic-bezier(0.25, 0.46, 0.45, 0.94)) and calm durations (3-6s)
  • Placement Strategy: Under headlines, in corners of containers, as section dividers, background textures
